# Mini Banking Application

## Description
The **Mini Banking Application** is a Java-based console application that simulates basic banking operations. It allows users to:
 Create new bank accounts  
 Deposit and withdraw money  
 Transfer funds between accounts  
 Check account balances  

This project demonstrates **core Java concepts** such as object-oriented programming, data handling, and user interaction. It is designed as a simple yet functional banking system for learning and practice purposes.

## Features
 Create and manage multiple bank accounts  
 Deposit and withdraw funds securely  
 Transfer money between accounts  
 View account details and balance  
 Simple console-based user interface

Language:** Java
